### 2020.06.26
#### 1158 요세푸스 문제
- 문제 자체를 이해하는데 어려웠지만, 알고나서는 쉽게 품
- 원형 연결리스트 이용
##### 만든함수
- list_insert : 리스트에 데이터 넣기
- list_next_node : 현재노드를 다음으로 이동하기
- list_remove : 현재노드를 지우고, 전에있는노드와 다음노드를 연결 
#### 1406 에디터
- 천천히 기능을 하나하나 구현함
- 더미 더블 연결리스트 이용
##### 만든함수
- list_init : 더미노드 만듬
- list_insert_tail : 리스트의 꼬리부터 데이터 넣기
- list_remove : 현재노드를 지우고, 전에있는노드와 다음노드를 연결 
- list_insert_cur : 현재노드와 전에있는노드 사이에 데이터 넣기
- cursor_move_left : 커서가 맨앞이 아니면 커서를 왼쪽으로 이동
- cursor_move_right : 커서가 맨뒤가 아니면 커서를 오른쪽으로 이동
- delete_char : 커서가 맨앞(더미노드)이 아니면 현재노드를 지움
- insert_char : 현재노드 다음에 데이터 넣기
- show_list : 리스트의 모든 데이터 보여주기
- free_all : 메모리가 할당된 모든 노드 메모리 해제
